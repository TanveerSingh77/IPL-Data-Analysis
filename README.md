# IPL Data Analysis Project

## Project Overview
This project is a **menu-driven IPL Data Analysis System** built using **Python** and designed to analyze historical **Indian Premier League (IPL)** data.

The project focuses on transforming raw cricket match data into meaningful insights through **data cleaning, exploratory data analysis (EDA), visualization, and statistical analysis**.

Using this project, users can analyze:
- Team performance
- Toss impact
- Top batters
- Top bowlers
- Player of the Match (MVP)
- Head-to-head team comparisons
- Key insights from historical IPL trends

This project demonstrates practical **data analytics workflow** on a real-world sports dataset.

---

# Problem Statement
IPL generates a massive amount of match data every season, but raw data alone does not provide meaningful insights.

The goal of this project is to:
- clean and structure raw IPL data,
- analyze important match patterns,
- identify team and player performance trends,
- and generate actionable insights from historical data.
